# Rhizoh — UI Language Guide (Translation Layer)

**Rol:** İç sistem dilini **kullanıcı diline** çevirmek — fazla teknik → korku; fazla mistik → ciddiyet kaybı; fazla oyun → Spiral ve marka zarar görür. Bu belge **copy, UI etiketleri ve marketing** için normatif çeviri tabanıdır.

**Stratejik not:** İleri sistemler çoğu zaman kendi terminolojisine aşık olur ve kullanıcıyı iç mimariye zorlar. Burada bilinçli bir **translation layer** tanımlanır — küçümsenmeyecek kadar önemli bir ürün olgunluğu. Üç katman çerçevesi: [RHIZOH_EMBODIED_PRODUCT_REALITY.md](RHIZOH_EMBODIED_PRODUCT_REALITY.md).

**İlişkili:** [Reference User Journey](RHIZOH_COMPANION_REFERENCE_JOURNEY.md) · [Personality stability](RHIZOH_COMPANION_PERSONALITY_STABILITY.md) · [Calm technology](RHIZOH_CALM_TECHNOLOGY_PRINCIPLE.md) · [Civilization framing §5](RHIZOH_CASTLE_CIVILIZATION_FRAMING.md)

## 1. Çeviri tablosu (iç → kullanıcı)

| İç sistem / spec | Kullanıcı dili (öneri) |
|------------------|------------------------|
| Witness (artifact) | **Memory Moment** · Verified moment |
| Chronicle | **Timeline** · Living memory |
| Castle Node | **Space** · Your place |
| Governance (PAG / policy) | **Safety Layer** · Ground rules |
| Replay (deterministik) | **Verified History** |
| Drift (ECDM / context) | **Context loss** (uyarı, panik dili değil) |
| TLOA snapshot | **Session snapshot** (power-user / destek; genelde gösterme) |
| Lineage | **Where this came from** (kısa, isteğe bağlı derinlik) |
| ETK / transition | **What changed** (özet) |
| HOGA / composition | *(genelde gösterme)* — “rules stayed consistent” hissi |

Power-user / geliştirici modu: ayrı yüzeyde teknik isimlere izin verilebilir; **Companion ana akışında** yukarıdaki dil.
