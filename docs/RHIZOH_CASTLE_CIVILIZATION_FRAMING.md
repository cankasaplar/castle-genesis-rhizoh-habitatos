## 3. Ürün sütunları (aynı çekirdek, beş yüzey)

1. **Rhizoh Companion** — sohbet, hafıza, chronicle, witness, harita, yaşam asistanı; **giriş kapısı**.  
2. **Castle Studio** — “IDE” değil hedef: **Reality Builder** (companion, ev düğümü, memory kuralları, automation ritüelleri, witness akışları — sürükle-bırak + derin bağlam).  
3. **Spiral (Collective Reality Simulation)** — “MMO” değil öncelikli framing: **Collective Chronicle** — koordinasyon, olay şahitliği, protest / tarih sürekliliği, ortak görevler, medeniyet chronicle’ı; *Civilization Sandbox*.  
4. **Robotics / embodied** — mimaride yer tutulur; public önce **Rhizoh Home Node** (spatial memory, oda şahitliği, bağlamsal otomasyon), sonra embodied agents.  
5. **Castle Node / smart environment** — **Personal Civilization Infrastructure**: kişisel hafıza alanı, AI kültürü, automation, epistemik kurallar, chronicle düğümü — klasik smart home’dan geniş.

## 8. Civilization Kernel Freeze (öneri)

Çekirdek invariant’lar ve epistemik borç **bilinçli dondurulduğunda**, ekip önceliği şuna kayabilir:

- UI / UX ve onboarding  
- Mobil ve süreklilik  
- Hafıza görselleştirmesi ve duygusal etkileşim  
- Gerçek kullanıcı döngüleri  

Soru kayması: *“Yeni invariant ekleyelim mi?”* → *“İnsanlar bu sistemde yaşayabilir mi?”*  

Freeze, çekirdeği öldürmez; **yüzey hızını** çekirdek hızından ayırır.
