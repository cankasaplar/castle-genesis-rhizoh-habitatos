## 2. Frozen core = launch ramp, not prison (Fırlatma rampası)

**Misread:** “Never changes” = ideological rigidity.  
**Intent:** **Immutable infrastructure** so upper layers can evolve without reality reference drift.

| Analogy | Frozen core in Rhizoh |
|---------|------------------------|
| Physics engine fixed → player feels free | v562–v570 execution subgraph + seal chain **stable** |
| Constitution fixed → society can create | EFIR-α layers + observation fabric **documented** |
| Rhythm fixed → music can improvise | Honest baseline + sovereign nodes **anchor** improvisation |

**Katılık, esnekliğin sigortasıdır.** Rigidity insures flexibility — not against it.

Expansion without dissolving ground: [`docs/LAYER_EXPANSION_PROTOCOL.md`](LAYER_EXPANSION_PROTOCOL.md) · habitat sprints — **not** silent edits to frozen `phase*.js` without stabilization graph.

Rhizoh **can** evolve. The sequence is fixed:

1. **Honest, rigid ground** (frozen execution floor + EFIR-α constitution)

3. **Habitat expansion** (research sprints, new nodes, new motors) — **never** upward causality from story to seal
