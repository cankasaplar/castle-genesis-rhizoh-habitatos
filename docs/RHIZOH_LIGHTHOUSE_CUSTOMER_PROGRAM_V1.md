# RHIZOH_LIGHTHOUSE_CUSTOMER_PROGRAM_V1

Kategori: Ilk müşteri programı (pilot motion)  
Hedef kitle: Field engineering, sales leadership, design partner ekipleri

## Program Goal

İlk lighthouse müşterilerle ölçülebilir şekilde "sovereign governance + proof routing" değerini kanıtlamak.

## Target Accounts (Strategic)

- Amazon Robotics ecosystem
- Ocado Group
- Symbotic
- ABB
- Siemens digital twin stack

## Pilot Offer

**90-day sovereign shadow deployment**

- Mevcut operasyonu kesmeden shadow telemetry ve proof gözlemi
- Advisory governance gates
- Haftalık risk/incident review
- Son hafta production enforcement readiness raporu

## KPI Framework

- Incident reduction
- Proof routing latency
- Replay coverage
- Governance intervention rate

## Success Criteria

- Pilot boyunca kritik olay sınıflarında düşüş eğilimi
- Policy ve proof akışında SLA/SLO uyumu
- Audit/replay taleplerinde operasyonel hızlanma
- Production phase için joint rollout planı

## Program Cadence

1. Week 0-2: Discovery + baseline  
2. Week 3-6: Shadow operations + tuning  
3. Week 7-10: Controlled enforcement  
4. Week 11-13: Executive readout + expansion plan
