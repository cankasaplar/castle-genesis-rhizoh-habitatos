## 0. Bu yolculuk neyi kanıtlar?

Kullanıcı **TLOA / HOGA / lineage** okumaz; şunları hisseder:

- geri dönmek istiyor mu?
- Castle’da “yaşam” kuruyor mu?
- zaman çizelgesi anlamlı mı?
- güven var mı?
- ekip (ileride) Studio’da kalıcı workflow kuruyor mu?

**Başarı artık yeni spec sayısıyla değil**, günlük kullanım ve **dönüş** ile ölçülür.

---

## 1. İlk 30 dakika (ilk temas)

| An | Deneyim (kullanıcı gözü) |
|-----|---------------------------|
| **İlk açılış** | Sessiz güven: tek duygusal merkez — Companion ([FREEZE-0 §2](RHIZOH_FREEZE_0.md)). |
| **Konuşma** | İsim / tercih öğrenme; aceleci “performans” değil, **sakin dinleme**. |
| **İlk Chronicle** | Küçük bir “anı” veya karar — **Timeline** dilinde ([UI guide](RHIZOH_UI_LANGUAGE_GUIDE.md)). |
| **İlk Castle node** | Kişisel **Space** hissi — boş şablon değil, “benim yerim” ([Castle runtime](CASTLE_NODE_RUNTIME_MODEL.md)). |

Teknik olarak arkada: kimlik, oturum sürekliliği, ilk mühür / hatırlatma — kullanıcı **motor** görmez.

---

## 2. İlk hafta (alışkanlık)

- **Hafıza hissi:** dün konuşulan şey bugün hatırlanıyor (surprise değil, **doğallık**).  
- **Günlük özet:** bildirim yağmuru değil; [Calm technology](RHIZOH_CALM_TECHNOLOGY_PRINCIPLE.md) ile özet.  
- **Witness moments** → kullanıcı dilinde **Memory Moment** ([UI guide](RHIZOH_UI_LANGUAGE_GUIDE.md)).  
- **Timeline** anlam kazanır; boş kalmaz, manipüle hissettirmez.  
- **Live continuity:** kopma sonrası “her şey uçtu” değil; [FREEZE-0 §1](RHIZOH_FREEZE_0.md) runtime sözü.

---

## 3. İlk ay (yaşam içine oturma)

- **Ev / çevre bağlantıları** (ürün kararı; mimari: node + izin).  
- **Takım node’ları** — paylaşılan Space, ortak Chronicle parçaları.  
- **Shared chronicle** — aile veya proje hattı.  
- **Routines** — küçük ritüeller; yine calm-first.  
- **World observe** — harita / dünya modu; Companion’dan **ayrı ekran** (tek merkez kuralı).

---

## 4. Disiplin geçişi (ekip için not)

**Çekirdek teorik fazın sonu** ile **gerçek ürün fazının başlangıcı** aynı değildir:

| Önceki ana savaş alanı | Şimdiki ana savaş alanı |
|------------------------|-------------------------|
| Epistemic closure, divergence, governance algebra | Onboarding, güven, dönüş oranı |
| | Auth / session / WS recovery ([FREEZE-0 §8](RHIZOH_FREEZE_0.md)) |
| | Uzun hafıza compaction, maliyet tavanı |
| | Gizlilik, moderasyon, çok cihaz |

“Tamamlama” hissi spec’lerle değil; **insanın geri gelmesiyle** doğrulanır.

---

## 5. Başarı soruları (checklist)

- İnsanlar Companion’a **geri dönüyor** mu?  
- Castle içinde **yaşam** kuruyor mu?  
- Timeline **gerçekten anlamlı** geliyor mu?  
- Sistem **güven** hissi veriyor mu?  
- (Phase D sonrası) ekipler Studio’da **kalıcı workflow** kuruyor mu?
