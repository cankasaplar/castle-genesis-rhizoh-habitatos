# EPISTEMIC BOUNDARY STATEMENT

Status: NORMATIVE / CONSTITUTION-LAYER  
Version: 1.0  
Date: 2026-05-09  
Change policy: append-only, explicit protocol required for any amendment

---

## 1. Canonical Truth

Canonical truth SHALL emerge only from attested artifact streams.

- raw runtime ≠ truth  
- media ≠ truth  
- model output ≠ truth  
- pointer ≠ truth  

## 2. External Boundary

External systems MAY inform observation.  
External systems SHALL NOT define canonical truth.

Scope: Media · NotebookLM · external sources · future model adapters  
Status: discovery-only · non-authoritative · provenance-labeled

## 3. Ledger Law

Artifact history is append-only.

- overwrite forbidden  
- correction = new artifact  
- causal chain preserved  

## 4. Projection Separation

Consumers project truth; they do not redefine truth.

Applies to: Castle Node Layer · Robotics Layer · future execution layers

## 5. Epistemic Category Freeze

No new epistemic category enters the system without explicit protocol definition.

## 6. Unknown Honesty

Unknown SHALL remain explicit.

- ignorance ≠ corruption  
- degraded ≠ unknown  

## 7. Authority Model

Observation is distributed.  
Execution authority is never inferred.

Applies to: Cursor · Claude · NotebookLM · all external observers

---

Rhizoh is a bounded epistemic protocol, not an unbounded execution authority.
